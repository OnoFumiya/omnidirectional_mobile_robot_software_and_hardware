# 背景

ロボットが自律的に人の生活を支援することが求められている[1]．
人にものを運んだり，人の荷物運搬を代替するなど幅広い活躍が期待できる．

それらの生活支援には，ロボットの柔軟な移動が不可欠である．
例えば物体を把持する場合に，ロボット全体が物体の把持姿勢に合わせることは，マニピュレーションに追加の自由度を持たせることができる．
